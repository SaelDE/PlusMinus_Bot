# PlusMinus_Bot

Twitch Chat-Bot with 1 custom command for a special voting method.

!plusminus
Checks the last 150 messages for:

\+\- \/ \-\+ \/ haugeNeutral \-\> Neutral

\+ \/ haugePlus -> Plus

\- \/ haugeMinus -> Minus

Output -> 'Ergebnis der letzten X MSGs: Plus: X --- Neutral: X --- Minus: X'
